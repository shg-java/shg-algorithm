## 栈
### 一、基本概念
栈是一种线性数据结构，遵循LIFO（Last In First out，后进先出）原则
```
栈顶 → │  4  │  ← 最后入栈，最先出栈
       │  3  │
       │  2  │
栈底 → │  1  │  ← 最先入栈，最后出栈
       └─────┘
```

二、核心操作
操作	说明	时间复杂度
push(e)	入栈，元素压入栈顶	O(1)
pop()	出栈，移除并返回栈顶元素	O(1)
peek()	查看栈顶元素，不移除	O(1)
isEmpty()	判断栈是否为空	O(1)
size()	返回栈中元素数量	O(1)
