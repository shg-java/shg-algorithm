### 回溯算法详解
一、核心思想
   回溯算法的本质是深度优先搜索（DFS）的一种，通过递归枚举所有可能的选择
当发现当前路径不满足条件时，撤销选择（回溯），返回上一步继续尝试其他路径
可以理解为在一颗决策树上的卞理过程
1.每一个节点代表一个状态
2.每条边代表一个选择
3.叶节点代表一个结果（满足条件则收集）
